# 终端命令自动审批规则

> 在终端执行命令时，自动处理交互式提示，无需用户手动确认

## 规则

1. **npm 安装确认**: 执行 `npm install` 时自动加 `--yes`（`npm install --yes`）
2. **npx 创建项目**: 使用 `--no-interactive` 或 `--yes` 标记避免交互
3. **Vitest 依赖**: 预装 `jsdom` / `happy-dom`，避免运行时提示安装
4. **Cargo 确认**: Rust 工具链无交互问题，但环境变量不足时自动处理（如 PATH 补全）
5. **用户不需要对终端提示作任何确认** — 所有应由 AI 自动处理
