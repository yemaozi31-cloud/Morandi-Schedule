# 手机端 TodayView 紧凑工具栏实施计划

## 当前状态

手机端 TodayView 目前：
- 番茄钟已压缩为 `[⏱ 开始专注]` 按钮
- 习惯卡片全尺寸展示在按钮下方，占半屏空间

## 用户需求

习惯按钮和番茄钟并排，习惯带气泡数字 + 收拉展开：

```
┌──────────────────────────────┐
│ 快速添加... NLP              │
├──────────────────────────────┤
│ [⏱ 开始专注] [🎯 习惯 3 ▼] │ ← 并排按钮，习惯带气泡+箭头
│                              │
│ ┌─ 阅读 ───── 🔥 3天 ────┐ │ ← 点击▼展开后才显示
│ │ ████████░░ 1/1次       │ │
│ │ [+ 打卡]               │ │
│ └─────────────────────────┘ │
│ ┌─ 运动 ───── 🔥 15天 ───┐ │
│ │ ████████░░ 1/1次       │ │
│ │ [+ 打卡]               │ │
│ └─────────────────────────┘ │
├──────────────────────────────┤
│ 今天任务列表                  │
└──────────────────────────────┘
```

交互行为：
- 点击 `[⏱ 开始专注]` → 导航到 /pomodoro
- 点击 `[🎯 习惯 3 ▼]` 的按钮主体 → 导航到 /habits
- 点击 `▼` 箭头 → 展开/收起习惯卡片列表（accordion）
- 气泡数字 = 今日待打卡的习惯数量

## 修改内容

### 文件: `src/views/TodayView.vue`

**模板修改**：
- 移除 `<div class="today-habits">` 的直接渲染
- 新增紧凑工具栏区域，包含：
  - 番茄钟按钮（复用现有 router-link）
  - 习惯按钮（含气泡 + 展开箭头）
- 新增可折叠的 habits-accordion 区域，v-if 控制展开/收起

**脚本修改**：
- 新增 `ref(habsExpand)` 控制展开状态
- 新增 `pendingHabitCount` computed：今日未完成的习惯数量

**CSS 修改**：
- 移除 `.today-habits` 的 `width: 100%` 无条件样式
- 新增 `.habits-accordion` 展开/收起过渡动画
- 手机端习惯按钮样式

### 桌面端同步修改

桌面端 right-panel 的今日习惯也改为可折叠：
- 默认展开（桌面有足够空间）
- 面板标题处加收起箭头

## 数据流

```
habitStore.checkIns → pendingHabitCount = today未完成的习惯数
  ↓
习惯按钮显示气泡数字
  ↓
点击▼ → habitsExpand = !habitsExpand
  ↓
habits-accordion v-if="habitsExpand" → 渲染 HabitCard 列表
  ↓
点击打卡 → habitStore.checkIn() → Toast
```

## 验证方式

1. 构建 0 错误
2. 手机端：两个按钮并排，气泡显示正确数字
3. 点击▼展开习惯卡片，再次点击收起
4. 点击习惯按钮整体 → 跳转到 /habits
5. 桌面端：右侧面板习惯区域也有折叠/展开功能
