# 持续事件 三链文档

## 1. 功能链：用户体验流程

```
关闭「持续事件」：
  截止日期: [____]  截止时间: [____]
  → 普通任务，只设 deadline

开启「持续事件」：
  开始日期: [____]  开始时间: [____]   ← 新增
  截止日期: [____]  截止时间: [____]   ← 不变
  → 时间段事件，从 开始 持续到 截止
  → 同一天：开始日期 = 截止日期
  → 跨多天：日历上区间内每天显示圆点
```

## 2. 数据链：字段映射

| 表单字段 | Task 字段 | 关闭时 | 开启时 |
|---------|----------|--------|--------|
| 截止日期 | `dueDate` | ✔ | ✔ (作为结束日) |
| 截止时间 | `dueTime` | ✔ | ✔ (作为结束时) |
| 开始日期 | `startDate` | null | ✔ |
| 开始时间 | `startTime` | null | ✔ |

## 3. 逻辑链：各视图处理

### 3.1 存储层 (taskStore.ts)

**createTask (L139-165)** ✅
- `startDate: data.startDate || null`
- `startTime: data.startTime || null`
- `isSpanning: data.isSpanning || null`

**updateTask ALLOWED (L187)** ✅
- 白名单包含 `'startDate', 'startTime', 'isSpanning'`

### 3.2 表单层 (TaskFormModal.vue)

**handleSave (L236-252)** ✅
- `startDate: form.isSpanning ? (form.startDate || null) : null`
- `startTime: form.isSpanning ? (form.startTime || null) : null`
- `endDate: form.isSpanning ? (form.endDate || null) : null`
- `endTime: form.isSpanning ? (form.endTime || null) : null`
- `isSpanning: form.isSpanning || null`

**watch editingTask (L268-282)** ✅
- `form.startDate = task.startDate || ''`
- `form.startTime = task.startTime || ''`

### 3.3 各视图 handleSave

所有视图的 handleSave 直接透传 data 到 taskStore：
- TodoView ✅ → `taskStore.createTask(data)` / `taskStore.updateTask(id, data)`
- TodayView ✅ → 同上
- InboxView ✅ → 同上
- CalendarView ✅ → 同上
- UpcomingView ✅ → 同上
- FloatingView ✅ → 同上

### 3.4 今日视图 (todayTasks)

- 持续事件：`today >= startDate && today <= dueDate`
- 普通任务：`dueDate === today`

### 3.5 列表视图 (TodoView groupedTasks)

⚠️ 未适配：持续事件按开始/截止日期区间放入正确分组

### 3.6 最近7天 (UpcomingView)

- nextDays/selectedTasks ✅：`dateStr >= startDate && dateStr <= dueDate`
- overdueTasks ✅：`isSpanning && startDate → dueDate < today`

### 3.7 日历 (CalendarView)

- taskMap ✅：遍历 `startDate → dueDate`
- dayTasks ✅：`selectedDay >= startDate && selectedDay <= dueDate`
- monthlyStats ✅：`startDate <= monthEnd && dueDate >= monthStart`

### 3.8 快速添加 (NLP)

⚠️ NLP 不支持解析 "9号到12号出差" 为 startDate/dueDate。当前只解析时间范围。
