## 二、功能模块详述与竞品对照

### 2.1 任务核心模块

| 功能 | 来源参考 | 说明 |
|------|---------|------|
| **自然语言输入** | Todoist, TickTick | 输入"明天下午3点买菜 p1 每天"自动解析 |
| **快速添加弹窗** | Todoist (Quick Add) | 全局快捷键 Ctrl+N 唤起 |
| **子任务多级嵌套** | Todoist | 最多 4 层嵌套 |
| **优先级 5 档** | TickTick | none / low / medium / high / urgent |
| **标签系统** | Things 3 | 支持筛选、彩色标签 |
| **重复规则** | Todoist | 每天/每周/工作日/每月/每年 + 自定义 |
| **提醒配置** | TickTick | 准时/提前N分钟/位置提醒(移动端) |
| **截止日期+时间** | 通用 | 可选日期+精确时间 |
| **备注/描述** | 通用 | Markdown 富文本 |
| **附件** | TickTick | 图片/文件 (后续) |

### 2.2 视图体系

| 视图 | 平台 | 设计思路 | 来源参考 |
|------|------|---------|---------|
| **今日聚焦** | 三端 | 聚合今日任务+过期任务+智能推荐 | Things 3 Today, MS To Do My Day |
| **艾森豪威尔矩阵** | 三端 | 重要/紧急四象限看板 | TickTick |
| **日历·月视图** | 桌面 | 标准网格，任务点标记 | TickTick, Google Calendar |
| **日历·周视图** | 桌面 | 横向时间轴，任务块显示 | Notion Calendar |
| **日历·日视图** | 桌面 | 垂直时间线，精确到小时 | Google Calendar |
| **移动端日历** | 移动 | 纵向逐日卡片列表，上下翻动 | **用户定制** |
| **看板视图** | 三端 | 按状态/优先级/标签分列 | Todoist, TickTick |
| **智能筛选器** | 三端 | `(today & p1) \| @urgent` 组合筛选 | Todoist Filters |

### 2.3 生产力工具

| 功能 | 来源参考 | 设计思路 |
|------|---------|---------|
| **番茄钟** | TickTick | 与任务绑定，专注时自动屏蔽干扰 |
| **白噪音** | TickTick | 内置专注音效（雨声/森林/咖啡馆） |
| **习惯追踪** | TickTick | 打卡+热力图+连续天数统计 |
| **统计看板** | Todoist (Karma) | 完成趋势图、番茄钟统计、习惯完成率 |

### 2.4 数据与交互层

| 功能 | 说明 |
|------|------|
| **本地优先** | IndexedDB 存储，离线可用，数据隐私 |
| **撤销重做** | 50 步历史记录，Ctrl+Z / Ctrl+Shift+Z |
| **键盘快捷键** | 全局+应用内，可自定义 |
| **悬浮窗** | 桌面端独有，快速查看/添加/完成 |
| **拖拽交互** | 排序、移动任务、调整日期 |

## 六、特色功能 — 竞品吸取与舍弃

### ✅ 吸取的功能

| 功能 | 来源 | 我们在体验上的改进 |
|------|------|------------------|
| 自然语言输入 | Todoist/TickTick | 支持中英文混合，更智能的模糊匹配 |
| 艾森豪威尔矩阵 | TickTick | **作为今日视图的默认布局**，而非独立页面 |
| 番茄钟+任务绑定 | TickTick | 悬浮窗模式下也可启动番茄钟 |
| 习惯追踪热力图 | TickTick | 莫兰迪色系热力图，视觉更温和 |
| 智能筛选器语法 | Todoist | 可视化构建+高级语法双模式 |
| 今日聚焦视图 | Things 3 | 聚合过期/今日/智能推荐三层 |
| 快速添加不弹窗 | Todoist | 悬浮窗底部直接输入 |

### ❌ 舍弃/弱化的功能

| 功能 | 来源 | 舍弃原因 |
|------|------|---------|
| **协作共享** | Todoist/TickTick | 个人工具定位，不做团队协作 |
| **位置提醒** | TickTick | 涉及后台定位权限，跨端复杂度高 |
| **语音输入** | TickTick | 需要额外语音识别服务，后续考虑 |
| **第三方集成** | Todoist (60+应用) | 个人项目，暂不做 API 开放 |
| **邮件创建任务** | Todoist | 需要邮件服务，超出个人工具范围 |
| **云端同步优先** | 所有竞品 | **改为本地优先**，云同步为可选功能 |

### 🎯 独特创新点

| 创新 | 说明 |
|------|------|
| **悬浮窗交互** | 桌面端独立迷你窗口，不干扰主工作流 |
| **莫兰迪全域主题** | 不只是换色，而是完整的设计语言系统 |
| **本地隐私优先** | 数据完全在本地，无需注册账号即可使用 |
| **三端统一代码** | Vue 3 一套代码运行于 Web/桌面/移动 |

## 八、主题系统设计

### 8.1 双主题模式

| 主题 | 风格 | 适用场景 |
|------|------|---------|
| **浅色暖调 (Morandi Light)** | 莫兰迪色系，暖灰米白，温和柔润 | 白天工作，长时间使用 |
| **深色冷调 (Morandi Dark)** | 黑色基底 + 莫兰迪点缀，低对比度 | 夜间使用，暗光环境 |

- 支持用户手动切换
- 未来可跟随系统自动切换

### 8.2 今日视图模式切换

今日视图支持在两种布局间切换（用户可自由选择）：

| 模式 | 布局 | 适合场景 |
|------|------|---------|
| **艾森豪威尔矩阵** | 四象限（重要/紧急 × 重要不紧急 × 紧急不重要 × 不紧急不重要） | 优先级管理、决策 |
| **时间线列表** | 过期任务高亮 → 今日任务 → 智能推荐（三层） | 快速执行、逐个完成 |

## 九、移动端 vs 桌面端差异化设计

| 维度 | 桌面端 (≥768px) | 移动端 (<768px) |
|------|----------------|----------------|
| **首页** | 今日聚焦 + 艾森豪威尔矩阵 | 今日聚焦列表 |
| **导航** | 左侧/顶部导航栏 | 底部 Tab 导航 |
| **日历** | 月/周/日 三视图切换 | 纵向逐日滚动列表 |
| **创建任务** | 快捷键 Ctl+N / 悬浮窗 / 按钮 | 浮动按钮 + 底部快速输入 |
| **番茄钟** | 侧边面板/悬浮窗 | 全屏专注模式 |
| **拖拽** | 鼠标拖拽 | 长按拖拽 |
| **悬浮窗** | Tauri 独立无边框窗口 | 无 |
